O foco aqui não é apenas "rodar código", mas sim consolidar uma base sólida em Python voltada para o processamento de dados em escala, integração com LLMs e arquitetura de pipelines.

## 🎯 Objetivo Profissional

Este repositório reflete o estudo aplicado em cenários de **Engenharia de Dados**, focando em:
1.  **Escrita de Código Limpo (Clean Code):** Seguindo o Zen of Python (PEP 20).
2.  **Eficiência de Algoritmos:** Escolha correta de estruturas para manipulação de grandes massas de dados.
3.  **Modularização:** Construção de funções reutilizáveis e sistemas desacoplados.
4.  **Versionamento Semântico:** Uso rigoroso de Git para controle de progresso e documentação de "commits".
